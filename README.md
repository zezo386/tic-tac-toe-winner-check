# tic-tac-toe-winner-check
a python script that determines either if X or O won or if it is a draw

# How to use
Enter a list with strings either with X or O 
the board goes as follows in indexes

0|1|2
- - -
3|4|5
- - -
6|7|8

so if i enter this list
["X","O","X","O","X","X","X","O","O"]
it will be interpreted in this format

X O X
O X X
X O O
